## Active Projects

### PINN Solver for the Nonlinear Schrödinger Equation
Implementing a physics-informed neural network in PyTorch that learns to solve
the NLSE with minimal training data by incorporating the equation's conservation
laws as soft constraints in the loss function.

- **Stack:** Python · PyTorch · NumPy · Matplotlib
- **Status:** In progress
- [View on GitHub](https://github.com/ahmad-sakib)

---

### Gradient Descent Visualizer
An interactive visualization of gradient descent on 2D loss surfaces, built to
develop intuition for optimization in machine learning. Features adjustable
learning rate, momentum, and several benchmark loss functions.

- **Stack:** JavaScript · Chart.js · HTML/CSS
- **Status:** Complete
- [Read the blog post](/blog/)

---

### Soliton Stability Simulator
High-performance numerical simulation of optical soliton propagation and
perturbation response using the split-step Fourier method in Julia.

- **Stack:** Julia · DifferentialEquations.jl · Plots.jl
- **Status:** In progress
